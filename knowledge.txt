5- definition le cahier des charges
Le cahier des charges, également appelé CDC, est un document élaboré lors du cadrage d’un projet il permet de formaliser les besoins, 
les objectifs, les contraintes, les fonctionnalités attendues, les délais et le budget prévisionnel de projet.

14- la matrice RACI

    la matrice RACI(Responsable, Accontable, Consulted, Informed) définit les roles et les responsabilités de chaque personne
    dans l'equipe de projet en clarifiant les attentes de participation et les information pour chaque taches ou activitées

    Responsable     => un personne qui realise la tache.
    Accountable     => qui approuve la tâche.
    consulted       => qui est consulté.
    informed        => qui doit etre informé.

diagramme de GANTT:
    le diagramme de Gantt est un outil de planifiaction de projet qui permet de visualiser les tâches, 
    les délais et les dépendances entre les tâches.
    il est représenté sous forme de barres  indiquent la durée de chaque tâche.
    cet outile aide a suivre l'avancement de projet d'une maniére graphique.

diagramme de PERT:
    Le diagramme PERT (Program Evaluation and Review Technique) est un outil de gestion de projet utilisé pour planifier et
    suivre les tâches complexes d'un projet. Il utilise un réseau de tâches pour représenter graphiquement les dépendances 
    et les relations entre les tâches. Le but du diagramme PERT est de déterminer la durée totale du projet en identifiant
    le chemin critique qui détermine le temps nécessaire pour terminer le projet
